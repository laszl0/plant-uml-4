Drawing UML with Plant UML

Language Reference Guide

(Version 8056)

PlantUML is an Open Source project that allows to quickly write:

  1. Sequence diagram,
  2. Usecase diagram,
  3. Class diagram,
  4. Activity diagram,
  5. Component diagram,
  6. State diagram,
  7. Object diagram,

Diagrams are defined using a simple and intuitive language.

Note: all data is taken from the manual of PlantUML v8056.
