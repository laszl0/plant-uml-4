2.7 Stereotypes

You can add stereotypes while defining actors and use cases using ”<<” and ”>>”.

            
                @startuml
                User << Human >>
                :Main Database: as MySql << Application >>
                (Start) << One Shot >>
                (Use the application) as (Use) << Main >>
                User -> (Start)
                User --> (Use)
                MySql --> (Use)
                @enduml
